Setup

To install the client framework perform the following steps:

npm install opensearch

This OpenSearch library requires Promises. If you are not shure whether you have it available use the following polyfill:

require('es6-promise').polyfill();

Testing

To run the unit tests do

npm test

To run the unit tests continuously do

npm run test:watch

Documentation

To generate the API documentation run:

npm run docs
